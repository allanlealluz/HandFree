Sobre o Projeto

O HandFree é uma plataforma inclusiva desenvolvida para ajudar pessoas com deficiência visual e motora a interagirem com dispositivos eletrônicos usando apenas a voz ou movimentos simples. Nosso objetivo é promover autonomia, independência e inclusão digital.

Funcionalidades

Comandos de Voz

Integração com assistentes virtuais e comandos personalizáveis.

Controle por Movimentos

Utilize webcam ou sensores para interagir de forma simples e intuitiva.

Integração IoT

Compatibilidade com dispositivos inteligentes e interface responsiva.

Como Usar

Para utilizar o HandFree, basta um navegador moderno com permissão de microfone ou webcam ativada. Em breve, disponibilizaremos uma versão beta para testes. Fique atento às novidades!
